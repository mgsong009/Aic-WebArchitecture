PI · Prompt Insight

UI · User Intervention
AI 초안에 대한 의미 있는 수정과 개입 정도를 측정합니다. 단순 복붙이 아닌 진짜 사고의 흔적을 포착합니다.
🌱
OI · Originality Index
주제 내에서의 자기 관점과 독창성을 평가합니다. 학생만의 시각과 창의적 사고를 계량화합니다.
🎯
AIC · AI Collaboration Index
PI, UI, OI를 종합한 최종 협업 지수입니다. AI를 도구로 잘 활용하는 역량을 하나의 숫자로 표현합니다.
📈
성장 추이 분석
과제별 시계열 분석으로 학생의 AI 활용 역량 성장을 추적합니다. 장기적 학습 패턴을 파악할 수 있습니다.
⚠️
위험군 조기 감지
